# Web Accessibility Practice

This repository is dedicated to practicing **web accessibility (a11y)** concepts through hands-on projects, based on courses and learning paths from **Alura**.

The main goal is to apply accessibility principles in real scenarios, focusing on inclusive and standards-compliant web development.

## Repository Structure

Each folder in this repository contains an **independent project**.

- Every project focuses on specific accessibility topics or techniques.
- Projects are self-contained and can be studied individually.
- The structure allows continuous expansion as new concepts are learned.
